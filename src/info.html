About KROHA.JS

KROHA.JS is an emulator of a first-generation von Neumann machine with a CISC (complex instruction set) architecture.

 Simulators like this one are used to study how a von Neumann machine works. Using it feels like sitting at a first-generation computer: it accepts data and returns results in binary, has a tiny amount of memory, only a few registers, and no operating system.

 The original "Krokha" is a DOS program, so the only way to run it on a modern computer is a DOS emulator or a virtual machine. That is inconvenient, and the age of the software causes plenty of problems along the way. KROHA.JS brings the same machine to the browser.

 KROHA.JS was written for educational purposes while learning JavaScript. It is based on a real program from 1995, with the amount of RAM increased to 16 cells.
